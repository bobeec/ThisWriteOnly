# BLOGthemeWP

**極限まで簡素化されたWordPressブログテーマ**

> 書くこと。読んでもらうこと。

note.comのように「書くこと」と「読むこと」だけを追求した、究極にシンプルなWordPressテーマです。

---

## ✨ 特徴

### 🎨 最小限のカスタマイズ

デザインできるのは **たった3つ** だけ：

1. **メインカラー** - リンクやボタンに使用されるアクセントカラー
2. **サイトアイコン** - ヘッダーに表示される小さなアイコン
3. **ヘッダー画像** - トップページに表示されるカバー画像

それ以外のデザイン要素は、すべて「読みやすさ」を最優先に最適化されています。

### ✏️ シンプルなエディター

WordPressブロックエディターを極限までシンプル化。使えるブロックは本当に必要なものだけ：

- 段落
- 見出し
- リスト
- 引用
- 画像・ギャラリー
- コード
- 動画・埋め込み

複雑なレイアウトブロックや装飾ブロックは非表示。**書くことだけに集中**できます。

### 🔍 SEO対応

プラグイン不要。テーマが自動で最適化：

- **メタディスクリプション** - 抜粋から自動生成
- **Open Graph** - SNSシェア用のメタタグ
- **Twitter Card** - Twitterカード対応
- **構造化データ** - JSON-LD（Article）
- **セマンティックHTML** - アクセシビリティ対応

### 🌙 ダークモード

OSの設定に応じて自動でダークモードに切り替わります。

### 📱 レスポンシブ

モバイルファーストで設計。どのデバイスでも最適な読書体験を提供します。

## 📦 インストール

### 方法1: ZIPアップロード

1. このリポジトリをZIPでダウンロード
2. WordPress管理画面 → 外観 → テーマ → 新規追加 → テーマのアップロード
3. ZIPファイルを選択してインストール
4. 有効化

### 方法2: FTP/SFTP

1. `BLOGthemeWP` フォルダを `/wp-content/themes/` にアップロード
2. WordPress管理画面 → 外観 → テーマ → BLOGthemeWP を有効化

---

## ⚙️ カスタマイズ

WordPress管理画面 → 外観 → カスタマイズ → **BLOGthemeWP 設定**

| 設定項目 | 説明 |
|---------|------|
| メインカラー | リンク・ボタンのアクセントカラー |
| サイトアイコン | 512×512px推奨。ヘッダーとファビコンに使用 |
| ヘッダー画像 | トップページ・アーカイブページに表示 |

## 🎯 デザイン仕様

### タイポグラフィ

- **本文**: 17px / 行間 2.0
- **フォント**: システムフォント（日本語最適化）
- **最大幅**: 680px（読みやすい文字数）

### カラーパレット

| 用途 | デフォルト |
|-----|----------|
| テキスト | `#292929` |
| サブテキスト | `#6b6b6b` |
| ミューテッド | `#999999` |
| 背景 | `#ffffff` |
| グレー背景 | `#fafafa` |
| ボーダー | `#e6e6e6` |
| プライマリ | `#1a8917` |

### 読みやすさへのこだわり

- 適切な行間（line-height: 2）
- 十分な段落間スペース
- コントラスト比 AAA 準拠
- 長文でも疲れにくいフォントサイズ
