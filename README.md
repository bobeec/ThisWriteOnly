# BLOGthemeWP

**書くこと。読んでもらうこと。**

極限まで簡素化されたWordPressブログテーマ。

---

## 特徴

- **シンプルな設定** - 表示/非表示のトグルのみ
- **シンプルなエディター** - 必要最小限のブロックのみ
- **シンプルなデザイン** - 白系背景で目に優しい
- **SEO対応済み** - メタタグ・OGP・JSON-LD自動出力
- **プラグイン不要** - テーマだけで完結

## インストール

1. ZIPをダウンロード
2. WordPress管理画面 → 外観 → テーマ → 新規追加 → アップロード
3. 有効化

## 使い方

1. **外観 → BLOGthemeWP** で表示項目を設定
2. **外観 → メニュー** でナビゲーションを作成
3. あとは記事を書くだけ

## 設定項目

| 項目 | 説明 |
|------|------|
| ヘッダー | サイトヘッダー全体 |
| サイトアイコン | ヘッダーのアイコン |
| サイトタイトル | ヘッダーのタイトル |
| ナビゲーション | メニュー |
| 投稿日 | 記事の投稿日 |
| 著者名 | 記事の著者 |
| 読了時間 | 「○分で読める」表示 |
| カテゴリー | 記事のカテゴリー |
| タグ | 記事のタグ |
| 著者ボックス | 記事下の著者情報 |
| 前後の記事リンク | 前/次の記事へのリンク |
| コメント | コメント欄 |
| フッター | サイトフッター |

## 推奨プラグイン

- **WP Multibyte Patch** - 日本語環境の最適化
- **XML Sitemaps** - サイトマップ自動生成
- **UpdraftPlus** - バックアップ
- **Akismet** - スパム対策

> SEOプラグインは不要です。テーマがメタタグ・OGP・構造化データを自動出力します。

## 動作要件

- WordPress 6.0+
- PHP 8.0+
